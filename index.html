You've enabled Auto-Cards support. Please follow these steps carefully:

  1. Go to the Auto-Cards GitHub Repository and copy the entire contents of its `Library.js` file.
  2. In AI Dungeon, go to the "Library" tab and paste the Auto-Cards code at the very top.
  3. Now, copy the code from the "Library" box below and paste it at the bottom of the "Library" tab in AI Dungeon, after the Auto-Cards code.
  4. For the "Input", "Context", and "Output" tabs, you can simply copy and paste the full contents from the boxes below.
